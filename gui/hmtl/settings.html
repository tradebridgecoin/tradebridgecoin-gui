Settings

Account the setting is on an account basis. therefore a valid Ethereum address is necessary for the settings

Ethereum Node (mainnet) a mainnet Ethereum Node for the live trading client to connect to, whether it be public node such as Infura or a private node running locally

Ethereum Node (testnet) an testnet Ethereum Node for the paper trading client to connect to, whether it be public node such as Infura or a private node running locally

Ethereum Node In Use choose the ethereum node to use

Starting Block Number the staring block number on the Ethereum blockchain, from which on transaction records will be retrieved. leave it empty if no historical transactions and/or positions need to be retrieved and listed

Referred by an Ehtereum address of the referrer. leave it empty if you are on your own
